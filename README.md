# Shellcode_Forensic_Android
Analyseur d'APK android

Sur internet il y a plein d'application android, on ne sais jamais si c'est fiable ou pas
TU peux aller sur apkpure mais te retrouvé avec un malware si tu ne vérrifie pas

Donc on fait un analyseur d'apk pour détecter les malware


Le lancer: 
chmod +x apk_analyzer.sh
./apk_analyzer.sh mon_app.apk
